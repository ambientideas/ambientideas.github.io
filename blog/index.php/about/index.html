I’m actively involved as an Open Source Software Architect for Java and Web Application projects across North, South, and Latin America.  I’m also acting as an Open Source evangelist, speaking about its benefits and practical applications in as many venues as I can.  Currently, my speaking engagements have included the Denver Open Source Users Group, Denver Java Users Group, Boulder Java Users Group, No Fluff Just Stuff tour, National Center for Atmospheric Research, Colorado Springs Open Source Users Group, and many large corporations. I’m also authoring a book on Presentation Patterns with Neal Ford and Nate Schutta due out in 2010.
